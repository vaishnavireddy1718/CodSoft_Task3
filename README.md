# 🥗 Restaurant Menu UI

## 📌 Overview

This project was created as part of the **CodSoft UI/UX Design Internship**.

The objective of this task was to design a modern and user-friendly restaurant menu interface that provides customers with a smooth food ordering experience through intuitive navigation, visually appealing layouts, and an organized menu structure.

- User-friendly layout
